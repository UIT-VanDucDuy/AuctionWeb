📁 CẤU TRÚC ADMIN PANEL
═══════════════════════════════════════════════════════════════════════════════

1. CONTROLLERS
   📄 src/main/java/com/example/auctionweb/controller/AdminController.java
   - Xử lý tất cả các chức năng admin
   - Endpoints:
     * GET  /admin                           - Dashboard tổng quan
     * GET  /admin/management                - Dashboard (alias)
     * GET  /admin/auction-registrations     - Danh sách đăng ký đấu giá
     * POST /admin/auction-registrations/{id}/approve  - Duyệt đăng ký đấu giá
     * POST /admin/auction-registrations/{id}/reject   - Từ chối đăng ký đấu giá
     * GET  /admin/product-registrations     - Danh sách đăng ký sản phẩm
     * POST /admin/product-registrations/{id}/approve  - Duyệt sản phẩm
     * POST /admin/product-registrations/{id}/reject   - Từ chối sản phẩm
     * GET  /admin/users                     - Danh sách users
     * POST /admin/users/{id}/activate       - Kích hoạt tài khoản
     * POST /admin/users/{id}/deactivate     - Khóa tài khoản

2. VIEWS (TEMPLATES)
   📂 src/main/resources/templates/admin/
   ├── management.html              - Dashboard tổng quan
   ├── auction-registrations.html   - Quản lý đăng ký đấu giá
   ├── product-registrations.html   - Quản lý đăng ký sản phẩm
   └── users.html                   - Quản lý users

3. STATIC RESOURCES
   📂 src/main/resources/static/
   ├── css/
   │   └── admin-style.css         - Style chung cho admin panel
   └── js/
       └── admin-common.js          - JavaScript functions chung

🔔 REAL-TIME NOTIFICATIONS
═══════════════════════════════════════════════════════════════════════════════

Khi admin thực hiện hành động (approve/reject), hệ thống sẽ:
1. Lưu notification vào database
2. Broadcast qua WebSocket đến user đang online
3. User nhận thông báo real-time tại /notifications.html

WebSocket Handler: NotificationWebSocketHandler.java
Endpoint: ws://localhost:8080/ws/notification

🚀 CÁCH SỬ DỤNG
═══════════════════════════════════════════════════════════════════════════════

1. Khởi động ứng dụng:
   .\gradlew.bat clean bootRun

2. Truy cập admin panel:
   http://localhost:8080/admin

3. Navigation:
   - Dashboard: /admin hoặc /admin/management
   - Đăng ký đấu giá: /admin/auction-registrations
   - Đăng ký sản phẩm: /admin/product-registrations
   - Quản lý users: /admin/users

4. Xem notifications real-time:
   http://localhost:8080/notifications.html
